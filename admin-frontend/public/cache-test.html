Timestamp:

If you see the current timestamp above, your browser is not caching this page.

To clear React app cache:

  1. Open http://localhost:4000/settings in your browser
  2. Press Cmd+Shift+R (Mac) or Ctrl+Shift+R (Windows/Linux) for a hard refresh
  3. Or open Developer Tools (F12), right-click the refresh button, and select "Empty Cache and Hard Reload"
